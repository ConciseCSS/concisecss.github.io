Using a combination of two classes, any element can be closed via another element. All that needs to be done is to have the .close class wrapped inside of an element containing the .close-this class.

There is no limit to what classes can be closed using the .close-this class, and absolutely any content can be put inside of the .close class.

This add-on requires the font-size() mixin in order for the SASS or LESS files to compile without modification, as as the jQuery JavaScript library.
