Breadcrumbs

Default

Breadcrumbs are a great way to let users know where they are at all times. The .breadcrumbs class adds spacing and slashes to each item in the breadcrumb trail.

This add-on requires the font-size() mixin in order for the SASS or LESS files to compile without modification.
