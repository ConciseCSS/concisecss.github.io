Blockquotes

The <blockquote> tag is used to specify text from another source.

There is no default styling for the <blockquote> tag by default in Concise, but we do have some code that will give it an indent with a left or right border and an italic font-style.

This add-on requires the font-size() mixin in order for the SASS or LESS files to compile without modification.

Preview

Augue purus et tortor dis egestas sociis dapibus montes? Adipiscing eu ac et est placerat? Odio tincidunt adipiscing lundium, et risus.

Lorem Ipsum
Code 
<blockquote>
  <p>Augue purus et tortor dis egestas sociis dapibus montes? Adipiscing eu ac et est placerat? Odio tincidunt adipiscing lundium, et risus.</p>

  <cite>Lorem Ipsum</cite>
</blockquote>

Reverse

The .blockquote-reverse class also can be added to any <blockquote> element to flip the direction of the blockquote indent and text from left to right.

Preview

Augue purus et tortor dis egestas sociis dapibus montes? Adipiscing eu ac et est placerat? Odio tincidunt adipiscing lundium, et risus.

Lorem Ipsum
Code 
<blockquote class="blockquote-reverse">
  <p>Augue purus et tortor dis egestas sociis dapibus montes? Adipiscing eu ac et est placerat? Odio tincidunt adipiscing lundium, et risus.</p>

  <cite>Lorem Ipsum</cite>
</blockquote>
