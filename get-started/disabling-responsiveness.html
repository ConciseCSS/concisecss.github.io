Disabling Responsiveness

Not interested in the responsive design that comes with Concise by default? Have no fear, disabling responsiveness can be accomplished with a few easy steps.

First, we need to ensure that the following <meta> tags are not present in our website's <head> section.

In our custom CSS files, or in <style> tags, we need to set a fixed width for our container element (I'm using 960 pixels as my example):

{% highlight css %}.container { width: 960px !important; }{% endhighlight %}

This overrides the Concise responsive container that is enabled by default. Next, we need to add a single class to our <body> tag, that will aid in disabling other responsive aspects of the site:

{% highlight css %}<body class="non-responsive">{% endhighlight %}
