Tooltips

Tooltips are the perfect way to display content when hovering over something. You can apply tooltips to any element as long as you are using one of the .tooltip-* classes coupled with the data-hint data attribute. Here I am using a simplified version of the Hint.css library.

Note: Tooltips take advantage of the HTML5 data attributes, so support is limited in older versions of Internet Explorer, Google Chrome, Firefox, Safari and Opera.
