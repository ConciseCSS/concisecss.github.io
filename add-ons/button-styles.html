Rounded

Adding the .border-radius class to any button can create a slight rounded corners effect.

Button

Button

Pill

Adding the class .pill to any button creates an extremely rounded button, similar to the shape of a pill.

Preview

Button

Button

Code

Borders

You can create bordered buttons by adding the class .btn-border to any element with the .btn class. There are also a host of colors that you can use to change the style of bordered buttons.

Social

These buttons are just an extension of the .btn class and rely on the Font Awesome icon font.

Preview
Dribbble Dropbox Facebook Flickr GitHub Google+ Instagram LinkedIn Pinterest Skype Tumblr Twitter YouTube
CSS 
/* Icon Buttons */
.btn.icon {
  font-family: "HelveticaNeue", "Helvetica Neue", Helvetica, Arial, sans-serif;
  font-weight: bold;
}

.btn.icon:before {
  font-family: FontAwesome;
  font-weight: normal;
  margin-right: 10px;
}

/* Dribbble */
.btn.icon.dribbble { background-color: #f36798; }
