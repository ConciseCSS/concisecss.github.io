Styling Scrollbars

WebKit-based browsers have a great property that allows you to style the way scroll bars look on the site with some very flexible pseudo-elements. Since these are only pseudo-elements, you can pick and choose where you want your custom scroll bars.

Simple Styling

By using these new pseudo-elements and some simple styling, you can create beautiful scroll bars for the body of your website.

Preview
Code {% highlight css %}::-webkit-scrollbar { width: 8px; } ::-webkit-scrollbar-track { background: #eee; border-radius: 5px; } ::-webkit-scrollbar-thumb { background: #aaa; border-radius: 5px; }{% endhighlight %}

Gradient Styling

Perhaps the flat, simple scroll bars aren't your style? Well here's a sample scroll bar that has a slight gradient.

Preview
Code {% highlight css %}::-webkit-scrollbar { width: 8px; } ::-webkit-scrollbar-track { background: #f3f3f3; border-radius: 5px; box-shadow: inset 0 0 6px rgba(0,0,0,0.2); } ::-webkit-scrollbar-thumb { background: #b3dced; background: -moz-linear-gradient(left, #b3dced 0%, #29b8e5 50%, #bce0ee 100%); background: -webkit-gradient(linear, left top, right top, color-stop(0%, #b3dced), color-stop(50%, #29b8e5), color-stop(100%, #bce0ee)); background: -webkit-linear-gradient(left, #b3dced 0%, #29b8e5 50%, #bce0ee 100%); background: -o-linear-gradient(left, #b3dced 0%, #29b8e5 50%, #bce0ee 100%); background: -ms-linear-gradient(left, #b3dced 0%, #29b8e5 50%, #bce0ee 100%); background: linear-gradient(to right, #b3dced 0%, #29b8e5 50%, #bce0ee 100%); border-radius: 5px; filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#b3dced', endColorstr='#bce0ee', GradientType=1); }{% endhighlight %}
