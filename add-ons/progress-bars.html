Progress Bars

A progress bar can be used to display a percentage of progress out of 100. The only two things driving these progress bars are an outer element with the .progress class as well as an inner <span> containing an inline style style="width: __%" where the "__" signifies a number under 100.

Thanks to Chris Coyier of CSS Tricks for his tutorial on CSS Progress Bars.

Striped & Animated

You can add the class .progress-striped to any progress bar to make it striped. In addition, adding the class .progress-animate will animate the stripes in the progress bar.
