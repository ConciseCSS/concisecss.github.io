Border Radius

If you wish to add a 4px border-radius around any element, you can do so with the .border-radius helper class.

Preprocessor Note: You can change the $border-radius variable in the _defaults.scss file to adjust the size of the border-radius.

Code 
<div class="border-radius">...</div>
