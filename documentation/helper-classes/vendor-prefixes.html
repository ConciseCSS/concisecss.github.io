Vendor Prefixes Mixin

In the /generic/_mixins.scss, we have a mixin that will allow you to easily write cross-browser compatible code for CSS3 features that have yet to be fully implemented. Browser prefixes included are:

  • Webkit Browsers (Chrome, Safari)
  • Internet Explorer
  • Firefox
  • Opera

Note: We suggest using a CSS post-processing tool such as Autoprefixer to add vendor prefixes to your CSS based on results from Can I Use.

Code 
@include vendor(border-radius, 4px);
