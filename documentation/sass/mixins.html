Mixins

Concise uses Sass mixins to help generate various features in the framework as well as provide extra functionality when working on projects.

All of these mixins can be found inside of the /helpers/_mixins.scss file.

Vendor Prefixes

In _mixins.scss, we have a mixin that will allow you to easily write cross-browser compatible code for CSS3 features that have yet to be fully implemented.

Parameters:

  • $property: what CSS property to generate vendor prefixes for
  • $value: the value of what was defined in $property
  • $vendors: what vendor prefixes to generate (default is none)

Note: We suggest using a CSS post-processing tool such as Autoprefixer to add vendor prefixes to your CSS based on results from Can I Use.

Code {% highlight css %}@include vendor(border-radius, 4px, webkit moz ms);{% endhighlight %}

Responsive Styles

In _mixins.scss, we have a mixin for handling media queries using the breakpoints in the $breakpoints-map map (and their respective names).

Code {% highlight css %}@include breakpoint(medium) { ... }{% endhighlight %}
