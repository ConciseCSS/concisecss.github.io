Vertical Navigation

Default

The .nav-vertical class allows you to stack navigation list items in a vertical manner.

Preview
  • Profile
  • Settings
  • Log Out
Code 
<nav class="nav-vertical">
  <ul>
    <li><a href="#">...</a></li>
  </ul>
</nav>

Adding a border

Simply adding the class .nav-bordered to your vertical navigation will add a border around the entire element as well as a hover effect for each list item.

Preview
  • Profile
  • Settings
  • Log Out
Code 
<nav class="nav-vertical nav-bordered">
  <ul>
    <li><a href="#">...</a></li>
  </ul>
</nav>

Adding a line

In the preview, .nav-line and .border-radius have both been added to our vertical navigation to add a border beneath our list items and add rounded corners, respectively.

Preview
  • Profile
  • Settings
  • Log Out
Code 
<nav class="nav-vertical nav-bordered nav-line border-radius">
  <ul>
    <li><a href="#">...</a></li>
  </ul>
</nav>

Floating content

You can easily float content to either side of a vertical navigation by using the .left or .right class.

Preview
  • Feed 20
  • Friends
  • Messages 5
  • Profile
  • Log Out
Code 
<nav class="nav-vertical nav-bordered nav-line border-radius">
  <ul>
    <li><a href="#">...</a> <span class="right badge">20</span></li>
    <li><a href="#">...</a> <span class="right badge bg-red">5</span></li>
    <li><a href="#">...</a> <span class="right icon chevron-right"></span></li>
  </ul>
</nav>

Dropdowns

jQuery dropdowns can be easily added to vertical navigation to display content. Dropdowns will display to the right of the vertical navigation menu by default.

Note: The icons used here are from a modified version of the Font Awesome icon font library.

Preview

Code 
<nav class="nav-vertical nav-bordered nav-line border-radius">
  <ul>
    <li><a href="#"><span class="icon th-list"></span> Feed</a></li>
    <li class="dropdown"><span class="icon user"></span> <a href="#">Profile</a> <span class="icon chevron-right right"></span>
    <ul class="dropdown-menu">
    <li>Settings</li>
    <li>Privacy</li>
    </ul>
    </li>
    <li><a href="#"><span class="icon power-off"></span> Log Out</a></li>
  </ul>
</nav>
