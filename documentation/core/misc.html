1.15Misc Styles

These are miscellaneous styles that have been added to Concise, but aren't really large enough to warrant their own page.

Print

We want your website to look great on the web, and when somebody tries to print it onto a piece of paper. This is why we have included some base print styles inside of src/core/layout/_print.scss. These are styles that will only display for printed pages.

You can change the base font-family for printed files by finding the $font-family-print variable inside of the /custom/_globals.scss file.

Horizontal Rule

You can use a horizontal rule element (<hr>) to separate content on a page.

Figures

You can use the <figure> tag to properly markup an image on your page.
