Emphasis

Small

The <small> tag and .small class can both be used to shrink text to a font size of 12 pixels. Note that heading elements receive their own font size for nested small elements.

Preprocessor Note: You can change the $small-size variable in the _variables.scss file to adjust the small font size.

The .micro class can be used to shrink text to a font size of 10 pixels.

Italics

The <em> tag is used for emphasizing text with italics.

Preview

This is a sentence with some italics in it.

Code {% highlight css %}This is a sentence with some italics in it.{% endhighlight %}

Note: The <i> tag also can be used for emphasizing text in an italic manner, although it is not recommended as <em> should be used. However, the <i> tag can still be used to define text in an alternative voice or mood.
