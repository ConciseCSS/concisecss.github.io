Font Weights

Three font-weight helper classes have been added for easy use with typography.

Class Font Weight
.weight-light 300
.weight-normal 400
.weight-semibold 600

Note: Only fonts that have the proper weights will be able to take advantage of these helper classes.

Code 
<h2 class="weight-light">...</h2>         
<h2 class="weight-normal">...</h2>
<h2 class="weight-semibold">...</h2>
