Addresses

The <address> tag is used to denote contact information for various parts of your web page.

For example, if an <address> tag is used within the <body> element, then the information contained within represents the contact information for the page on which it is contained.

However, the <address> tag also can be used within HTML5's <article> element, in which case the information contained within represents the contact information for the article on which it is contained.

Note: Due to the default styling of the <address> tag in browsers, line breaks must be placed after each line if a <p> tag is not used.
