Container

Our entire design is held in place with the .container class. We use the container to possess the elements of our design within a specific width, and also adjust the width at specific breakpoints so as to provide compatibility for mobile devices.

Here is a breakdown of the width of the .container class at various viewports:

Extra Large
≥ 1280px		 Large
≥ 1120px		 Regular
≥ 960px		 Small
≥ 768px		 Extra Small
≤ 768px
Width 1140px 960px 768px 620px 90%

Of course, it is extremely easy to modify the width of the .container class at the various breakpoints. Just copy the code below into your custom CSS file and adjust the widths as you see fit.

Code 
.container { width: 90%; }

@media (min-width: 48em) /* 768px */ { .container { width: 620px; } }

@media (min-width: 60em) /* 960px */ { .container { width: 768px; } }

@media (min-width: 70em) /* 1120px */ { .container { width: 960px; } }

@media (min-width: 80em) /* 1280px */ { .container { width: 1140px; } }
