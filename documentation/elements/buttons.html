Dropdown

Dropdowns can be added to button elements very easily. Below is an example of a button dropdown using a handful of the helper classes that are provided by Concise.

Note: The icons used here are from a modified version of the Font Awesome icon font library.

Preview
Dropdown
  • Profile
  • Settings
  • Log out
Code 
<div class="btn dropdown line" tabindex="1">
    Dropdown <span class="caret"></span>
	<ul class="dropdown-menu">
		<li><a href="#">Profile <span class="icon user right"></span></a></li>
		<li><a href="#">Settings <span class="icon cog right"></span></a></li>
		<li><a href="#">Log out <span class="icon times right"></span></a></li>
	</ul>
</div>

Centering

As the .btn class is an inline-block element, simply adding the class .center will not suffice in actually centering the element.

Instead, we must wrap the element inside the .text-center class, which will provide proper centering of our button.

Preview
Center Button
Code 
<div class="text-center">
    <a class="btn">Center Button</a>
</div>

Collapsable

By default, all buttons maintain their default width as the browser viewport scales down. However, adding the class .btn-collapse-full allows the button to take up 100% the width of the screen after the browser has reached the extra-small breakpoint (which is 30em by default).

Preview
Collapsable Button
Code 
<div class="text-center">
    <a class="btn btn-collapse-full">Collapsable Button</a>
</div>
