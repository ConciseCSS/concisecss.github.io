Forms

Basic

This is a basic form composed of the <form> tag containing various <label>, <input>, <select> and <textarea> fields.

Each label and input pair are wrapped in a .form-item class that adds bottom margin and keeps our respective labels and inputs in their own block so no content wraps around them.

Preprocessor Note: You can change the $placeholder-color variable in the _defaults.scss file to adjust the placeholder text color.

<form>            
  <div class="form-item">
    <label for="name">Name:</label>
    <input type="text" name="name" placeholder="John Doe" />
  </div>

    <label for="email">Email:</label>

    <input type="text" name="website" placeholder="http://example.com" disabled />

    <label for="gender">Gender:</label>

    <input type="radio" name="gender" value="Female" /> Female

Fieldset

Wrapping your form content with a <fieldset> tag allows you to group related elements inside of your form. Adding the <legend> tag allows you to denote each section with a small heading.

Preview
User Information
Code 
<form>
  <fieldset>
    <legend>...</legend>
  </fieldset>
</form>

Inline Form

Adding the class .form-inline to your <form> element allows you to display your form elements in an inline fashion. By default, the .form-inline class will give each <label> element a width of 100px.

Preview
Code 
<form class="form-inline">
  ...
</form>

Required Fields

Adding the class .required along with the HTML5 field attribute required to a form element will help stylize required fields. Any field will display a red border when focused on or filled out improperly. If the field is filled out correctly, that border will become green.

Note: This example is using the .screen-reader class to hide the <label> tag. This is good for accessibility if you wish to not display field labels.

Preview
Code 
<form class="form-inline">              
  <div class="form-item">
    <label for="name" class="screen-reader">Name:</label>
    <input type="text" class="required" name="name" placeholder="John Doe" required />
  </div>
</form>

Rounded corners

Rounded corners can be added to form fields two different ways. First, you can add the class .border-radius to the <form> tag, which will render all form fields with rounded corners. However, you also can apply the class to individual form fields if you wish for only specific fields to have rounded corners.

Preview
Code 
<form class="form-inline border-radius">              
  ...
</form>
