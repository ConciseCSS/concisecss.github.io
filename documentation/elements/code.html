Code

Inline

Wrap inline snippets of code using the <code> tag.

Preview

Use the .container class for containing the elements of our design.

Code 
Use the <code>.container</code> class for containing the elements of your design.

Blocks

Large blocks of code should be contained within the <pre> tag.

Preview 
.container {
  margin: 0 auto;
  width: 1080px;
}
Code 
<pre>.container {
  margin: 0 auto;
  width: 1080px;
}</pre>

Blocks with Background

If you would like a more stylized code block with a background, you can use a <code> tag wrapped inside of a <pre> tag.

Preview 
.container {
  margin: 0 auto;
  width: 1080px;
}
Code 
<pre><code>.container {
  margin: 0 auto;
  width: 1080px;
}</code></pre>

User Input

You also may wish to use the <kbd> tag to indicate content that was entered via the keyboard.

Preview

Use the ls command to list the contents of a directory on a UNIX machine.

Code 
<p>Use the <kbd>ls</kbd> command to list the contents of a directory on a UNIX machine.</p>
